# Gratis gebruiken op je telefoon

## Snelste gratis manier
1. Maak een gratis GitHub-account.
2. Maak een nieuwe repository, bijvoorbeeld `boekenhoekje`.
3. Upload **alle bestanden uit deze map**.
4. Ga in GitHub naar **Settings > Pages**.
5. Kies **Deploy from a branch** en selecteer **main / root**.
6. Na ongeveer een minuut krijg je een gratis link.
7. Open die link op je telefoon en kies **Zet op beginscherm / Add to Home Screen**.

## Nog makkelijker voor testen
- Ga naar Netlify Drop.
- Sleep de bestanden of de zip erop.
- Je krijgt meteen een gratis link.

## Belangrijk
- Omdat dit een browser-app is, blijven je gegevens alleen bewaard op hetzelfde apparaat en dezelfde browser.
- Voor een echt permanente online database is later iets extra's nodig, maar voor jouw doel werkt dit gratis prima.
